## Tecnologias usadas:
* HTML
* CSS
* JS
* PYTHON

## Python libs:
### Flask:
Criar o servidor do chatbot, definir rotas, renderizar o HTML e enviar/receber as mensagens, realizando a comunicação front(HTML/JS) e backend(Python).

### DateTime:
Usado para obter a hora atual, que depois é exibida pelo chatbot.

### Pytz:
Esta biblioteca permite trabalhar com fusos horários corretamente, garantindo que a hora mostrada pelo chatbot seja a de brasília independente do fuso do servidor.
